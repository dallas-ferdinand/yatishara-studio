# Cut types

| ID | Cut | Use |
|----|-----|-----|
| `cut:straight` | Hard cut on action complete | Default |
| `cut:match-action` | Hand reaches → cut to object | Geography link |
| `cut:time-passage` | Object same, light changed | Joe time |
| `cut:jump-time` | Same frame years later | Rare 180s |
| `cut:hold-no-cut` | Single shot scene | Witness moment |
| `cut:J-cut` | Sound leads next shot | Post assembly note |
| `cut:L-cut` | Image leads | Post assembly note |
