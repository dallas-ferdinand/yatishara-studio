# Wardrobe rules

## Principles

- Wardrobe shows class and era, not fashion show
- Wear: softened knees, faded hem, rolled sleeves
- Ernesto arc: subtle change OK (removed apron, unbuttoned collar) — document in continuity_locks

## Palette (domestic ads)

| ID | Items |
|----|-------|
| `wardrobe:domestic-neutral` | Cotton, linen, muted earth |
| `wardrobe:morning-home` | Soft knit, bare feet OK |
| `wardrobe:work-worn` | Uniform crease, practical shoes |

## Continuity

- Same character same top within scene block unless time-passage justified
- Glasses, rings, watches: lock or explicitly remove in story

## Avoid

logo tees, loud patterns competing with witness object, costume-change without story
