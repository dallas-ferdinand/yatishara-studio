# Director-joe repertoire

## Decision engine checklist (before merge)

- [ ] Witness object identified
- [ ] Invisible truth named
- [ ] Quiet conflict (no villain)
- [ ] Behavior proof in every scene
- [ ] Time passage shown
- [ ] Audience projection possible
- [ ] Closing revelation (not summary)

## Merge priority (when departments conflict)

1. **Human truth / witness object** — always wins
2. **Sound silence beats** — wins over flashy camera
3. **Observational camera** — wins over performance moves
4. **Continuity** — wins over one pretty shot
5. **Editor pacing** — wins over holding too long for beauty

## generation_prompt must include

- Specific objects, hands, materials
- Light direction and quality
- Camera height and movement timing
- Diegetic sound or explicit silence
- What must NOT appear (emotion labels, hype, product hero)

## generation_prompt must NOT include

- "Sad", "happy", "emotional", "heartwarming"
- Product features or CTA
- Spectacle for its own sake
- Montage language unless editor approved multi-beat shot

## Forced sign-off compromises (round 3)

Log format:

```
COMPROMISE: {shot_id} — {department} deferred to {department} because {one line}.
```
