# Prop master repertoire

## Material vocabulary

| ID | Description |
|----|-------------|
| `mat:worn-glass` | Dull glass, smudges, not studio clean |
| `mat:chipped-enamel` | Mug, pot, aged kitchen |
| `mat:sun-faded-wood` | Table, chair, porch |
| `mat:creased-linen` | Cloth props |
| `mat:wax-residue` | Jar lids, candles |

## Wear levels

| ID | Level |
|----|-------|
| `wear:light` | Daily use |
| `wear:medium` | Visible history |
| `wear:heavy` | Generational object |
