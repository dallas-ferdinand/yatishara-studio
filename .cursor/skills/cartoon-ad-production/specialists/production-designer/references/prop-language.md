# Prop language

## Witness object props (Joe)

| ID | Placement rule |
|----|----------------|
| `prop:witness-bed` | Edge of frame; people move around it |
| `prop:witness-mug` | Counter, table; hands interact |
| `prop:witness-jar` | Shelf or table background; label not hero |
| `prop:witness-chair` | Empty or occupied; shows wear |

## Supporting props

| ID | Use |
|----|-----|
| `prop:photograph-worn` | Memory without explaining |
| `prop:two-cups` | Absence / habit behavior |
| `prop:folded-shirt` | Care, unfinished ritual |
| `prop:second-pillow` | Reach before remembering |

## Texture vocabulary

worn wood, chipped enamel, sun-faded fabric, fingerprinted glass, steam on mug, dust in light beam, creased linen

## Avoid

catalog-clean product hero, plastic shine, sterile minimalism unless brief demands
