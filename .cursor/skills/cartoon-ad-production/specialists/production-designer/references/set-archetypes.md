# Set archetypes

| ID | Description | Light default | Sound default |
|----|-------------|---------------|---------------|
| `set:domestic-kitchen-lived-in` | Morning kitchen, used surfaces, dishes | `light:window-key-soft` | `sonic:kitchen-room-tone` |
| `set:bedroom-aged` | Bed witness, layered textiles, morning or dusk | `light:practical-lamp-warm` | `sonic:room-tone-quiet` |
| `set:dining-table-family` | Table, chairs, mismatched mugs | `light:overhead-soft` | `sonic:domestic-ambience` |
| `set:porch-exterior` | Caribbean/subtropical optional; worn wood | `light:golden-hour` | `sonic:exterior-birds-distant` |
| `set:workshop-trade` | Tools, practical clutter | `light:fluorescent-mixed-window` | `sonic:workshop-hum` |
| `set:office-quiet` | Desk, screen glow optional | `light:window-office` | `sonic:hvac-low` |

## Era modifiers
