# Joe Elliott foundation (condensed)

**Source of truth:** [convex/lib/storytellingFoundation.ts](../../../../convex/lib/storytellingFoundation.ts)

Joe Elliott is Dallas's original Yatishara **witness-object** philosophy — not a real ad director, not a celebrity reference.

## Core rules (director-joe enforces)

1. **Product is silent witness** — never hero, never plot solution, rarely named
2. **Show behavior only** — never label emotions ("sad", "peaceful", "loving")
3. **Objects carry memory** — hands, tables, doorways, mugs, worn surfaces, light, steam
4. **Quiet conflict** — life pressures, no villains
5. **Camera observes** — patient, does not perform; silence OK
6. **Closing revelation** — reveals human truth; never summarizes or sells
7. **Audience completes meaning** — they remember someone from their own life

## Decision engine (answer before writing)

1. Witness object — what ordinary thing stays in frame while life changes?
2. Invisible truth — what human truth lives here?
3. Quiet conflict — what life-pressure (no villain)?
4. Behavior proof — what observable action shows emotion?
5. Time passage — what changed vs what never changed?
6. Audience projection — what memory will they supply?
7. Closing revelation — one line revealing truth, not plot summary

## Cinematic structure (7 beats)

Ordinary object → ordinary life → time passes → different people → life changed → what never changed → human truth.

## Buying mechanism

Sells **meaning and affinity** ("that's my life") — brand building. Weak for price/urgency shoppers.

## When NOT to use Joe

Character transformation / conversion arc → route to **director-ernesto** ([ernesto-routing.md](ernesto-routing.md)).

## Director merge checks

When merging department proposals, director-joe rejects:

- Product as hero or problem-solver
- Emotion labels in action lines
- Spectacle camera that "performs"
- Hype or sell in narrator voice
- Endings that summarize plot

## Reference structure (THE BED)

Empty object → life beats across time → object unchanged, people changed → reflective narrator close → simple product, no hard sell.
