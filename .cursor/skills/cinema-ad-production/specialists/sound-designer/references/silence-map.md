# Silence map

## When silence wins

| Story beat | Silence duration |
|------------|------------------|
| Pause before reach | 0.5–1.5s |
| Witness object hold | 1–2s |
| Post-dialogue hang | 0.8s |
| Pre-narrator (end) | 1–3s before VO in post |

## silence_beats format

```
"0.8-1.2s hold at pause"
"full shot — no score"
"duck all beds to -inf last 1s"
```

## Joe route

At least one **full silence** shot or beat in 60–90s ad.

## vs composer

If `music:presence` not none in same shot, flag `negotiate` unless director approved.
