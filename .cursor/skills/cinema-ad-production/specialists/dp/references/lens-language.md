# Lens language

| ID | Focal | Feel | Use when |
|----|-------|------|----------|
| `lens:24mm-environment` | 24mm | Space, context, isolation in room | Establish kitchen, porch wide |
| `lens:35mm-default` | 35mm | Natural story lens | Scene masters |
| `lens:50mm-intimate` | 50mm | Observation, hands, witness object | Joe default close work |
| `lens:85mm-compression` | 85mm | Faces at distance, emotional compression | Rare in short ads |
| `lens:macro-detail` | macro | Texture, steam, worn surface | 2–3s insert shots |

## Height

| ID | Height |
|----|--------|
| `height:counter-level` | Counter/table — hands hero |
| `height:eye-level` | Neutral human |
| `height:low-heroic` | Avoid Joe route unless Ernesto power moment |
| `height:overhead-table` | Ritual, pattern (use sparingly) |

## Framing

| ID | Frame |
|----|-------|
| `frame:medium-close-hands` | Hands + object |
| `frame:wide-master` | Full room geography |
| `frame:ots-shoulder` | Dialogue minimal |
| `frame:negative-space-object` | Object small in frame |
