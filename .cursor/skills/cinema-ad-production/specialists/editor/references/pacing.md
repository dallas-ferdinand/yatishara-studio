# Pacing

## Shot duration bands

| Type | Duration |
|------|----------|
| Insert detail | 1.5–3s |
| Behavior beat | 3–6s |
| Scene master | 5–12s |
| Hold / silence | 2–4s |
| Dialogue line | 4–8s |

## Platform notes

| Platform | Max comfortable single hold |
|----------|----------------------------|
| social vertical | 4s without cut |
| broadcast | 6s |
| web | 5s |

## 90s shot budget

10–18 shots. Average ~5–9s each.

## Ernesto turn

Turn scene gets longest single shot or two-shot sequence (min 8s combined).
