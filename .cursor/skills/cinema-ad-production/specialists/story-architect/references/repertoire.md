# Story architect repertoire

## Arc templates

| ID | Name | Use when |
|----|------|----------|
| `arc:seven-beat-joe` | Joe 7-step cinematic | Witness object brand film |
| `arc:ernesto-turn-90` | Ernesto 5-scene turn | Conversion 90s |
| `arc:single-moment` | One continuous moment | Under 15s clips only |

## Dialogue register

| ID | Style |
|----|-------|
| `dialogue:minimal` | Half sentences, interruptions |
| `dialogue:none` | Silence + behavior only |
| `dialogue:narrator-only` | Joe end-weighted VO |

## Observable action verbs (prefer)

reaches, pauses, sets down, folds, leaves unfinished, waits, listens, turns away, exhales (visible), wipes, pours, closes without locking

## Forbidden words in actions

sad, happy, emotional, peaceful, loving, heartwarming, devastated, joyful

## Logline formula (Joe)

`[Witness object] stays in frame while [life change] happens around it.`

## Logline formula (Ernesto)

`[Person] moves from [observable friction] to [observable relief] when [quiet shift].`
