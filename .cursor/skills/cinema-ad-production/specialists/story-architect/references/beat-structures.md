# Beat structures by duration

**Mandatory companion:** [../../../references/timing-foundation.md](../../../references/timing-foundation.md) — three clocks, tier budgets, ASL targets.

## 15s — social punch (1 scene)

| Segment | Duration | Joe beats | Ernesto beats |
|---------|----------|-----------|---------------|
| Hook | 0–4s | Witness object in frame + geography | Friction visible in first 3s |
| Proof | 4–10s | **One** observable behavior | Pressure micro-beat |
| Peak | 6–8s | Hands pause / ritual incomplete | Behavior shift |
| End anchor | 10–15s | Truth line or object hold + CTA | Relief proof |

**Constraints:** 1 scene, 1 location, 3–5 shots, 3–4 inferential beats max. **No** time-passage montage. Narrator: 0–1 line at end.

## 30s — social standard (1–2 scenes)

| Scene | Duration | Joe | Ernesto |
|-------|----------|-----|---------|
| SC01 | 18–22s | Hook + friction + peak | Friction + pressure + turn |
| SC02 | 8–12s | End anchor + close | Relief + CTA |

Or **single scene** 30s continuous. 5–7 shots. Peak at ~14–17s.

## 60s — compressed

| Scene | Duration | Joe beats | Ernesto beats |
|-------|----------|-----------|---------------|
| SC01 | 15s | Object + ordinary moment | Friction visible |
| SC02 | 20s | Life pressure | Pressure continues |
| SC03 | 15s | Time hint | Turn begins |
| SC04 | 10s | — | Relief proof |
| SC05 | 0s | Close (in SC04) | Close |

Total: 4 scenes, ~60s. 8–12 shots in Phase C. Peak ~33–39s.

## 90s — standard

| Scene | Duration | Joe (7-beat map) |
|-------|----------|------------------|
| SC01 | 12s | Ordinary object |
| SC02 | 18s | Ordinary life |
| SC03 | 18s | Time passes |
| SC04 | 18s | Different people |
| SC05 | 15s | Life changed |
| SC06 | 9s | What never changed + close |

Ernesto: 5 scenes — friction (20s) → pressure (25s) → turn (25s) → relief (15s) → close (5s). Peak ~54–63s.

## 180s — expanded

8 scenes. Allow 2–3 time-passage beats. Minimum 4s per observable action beat. 16–24 shots.

## Scene writing rules (all tiers)

- One main observable action per **4–6 seconds** of screen time (15s: per **3–4s**)
- Dialogue: max 1–2 lines per scene unless brief demands (15s: max 1 line total)
- Narrator: single block at end only (Joe); Ernesto may omit narrator
- `sum(scenes.duration_sec)` = brief target **±2s**

## Tier selection at intake

| User language | Default tier |
|---------------|--------------|
| bumper / TikTok / Reels | 15s |
| social / feed / pre-roll | 30s |
| broadcast / TV spot | 60s |
| brand film (default) | 90s |
| long form | 180s |

Do not plan 90s scene structure for a 15s brief.
