# Director-ernesto repertoire

## Character turn beat sheet (90s example)

| Beat | Duration | Content |
|------|----------|---------|
| Friction visible | 0–25s | Observable strain behavior |
| Pressure continues | 25–50s | Same friction, new context |
| Turn begins | 50–70s | Small behavior shift |
| Relief proof | 70–85s | Tomorrow lighter — observable |
| Close | 85–90s | One line forward, not memory |

Scale beats for 60s (compress middle) or 180s (expand pressure + turn).

## Arc types

| ID | Name | Opening | Closing |
|----|------|---------|---------|
| `arc:permission` | Permission granted | Hesitation, withheld action | Acts freely |
| `arc:burden-lift` | Burden lighter | Overloaded routine | One task easier |
| `arc:first-time` | First time OK | Avoidance | Tries once |
| `arc:return` | Return to self | Diminished self | Small reclaim |

## Merge priority (Ernesto)

1. **Character turn clarity** — wins over beauty
2. **Friction readability** — wins over subtlety if ambiguous
3. **Forward close** — wins over nostalgic image
4. **Editor pacing** — platform duration hard cap

## generation_prompt (Ernesto)

Include observable behavior state per shot: what the character **does**, not feels.

Example: "She sets the phone face-down without checking" not "She feels relieved."
